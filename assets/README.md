## Логотип (уже в проекте ✓)

Экспортирован из брендбука и подключён в шаблоны:

| Файл | Что это | Где используется |
|---|---|---|
| `flow-logo-white.svg` | вордмарк `FLOW` белыми буквами + градиентный завиток-«O» | подпись/подвал на **тёмной** теме |
| `flow-logo.svg`       | тот же вордмарк тёмными буквами (`#242427`) | на **светлой** теме |
| `flow-mark.svg`       | только завиток-волна (он же «O») | иконка/аватар профиля (`.logo-mark`) |

Переключение тёмная/светлая идёт автоматически через токен `--logo-src` (см. `css/tokens.css`).

## Фото

В слоты сейчас стоят **стоковые заглушки** (Pexels, бесплатная лицензия) — `stretch.jpg`,
`fitness.jpg`, `dance.jpg`, `trainer.jpg`, `face.jpg`. Замени на реальные снимки студии
(те же имена — и менять разметку не нужно), либо подключи новые так:

```html
<div class="photo" style="background-image:url('assets/nastya.jpg')"></div>
```

> ⚠️ Путь задаём через `background-image` (резолвится от HTML-файла). НЕ через `--photo` —
> там url() резолвится относительно `css/` и путь ломается (`css/assets/...`).
> Сила фирменного дуотона: класс `duo` / `duo--light` / `duo--strong`.

Стоки — для демонстрации; перед публикацией заменить на свои фото (права на стоки Pexels
свободные, но свои живые кадры студии всегда сильнее).
