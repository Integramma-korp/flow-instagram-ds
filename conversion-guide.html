Читаемость и конверсия

Короткий свод: как сделать, чтобы оффер прочёлся за секунду и вёл на запись. Читаемость — это первый рычаг конверсии.

Читаемость

  • Над фото текст всегда светлый (--on-photo) + скрим, в любой теме.
  • Градиентное слово по фото — тень через filter: drop-shadow() (обычная не работает).
  • Критичное (время · цена · CTA) — только Jost, не сериф по фото.
  • Минимальный кегль ≥34px (в 1080). Жирно и контрастно.
  • Максимум 10–15 слов на слайде. Больше — теряется за 3 секунды листания.
  • Контрастность ≥4.5:1 (WCAG). Белый по скриму это всегда выполняется; риск — серый или лиловый текст без скрима.
  • Не ставить важное в safe-zone IG: верх 250 / низ 420px.
  • Не пастель по фото — washed out.

Конверсия

  • Голубой #08CEE9 — только запись на пробное (плитка · таплинк · хайлайт «с чего начать»). Больше нигде.
  • Один доминантный CTA на макет, остальное тише.
  • Цена — крупнейший Jost после заголовка; ₽ меньше цифры.
  • Действие — в нижней трети (зона большого пальца).
  • Ургентность точными числами: «осталось 3 из 8 мест», «набор закрывается в пятницу» — конкретика бьёт «ограниченное предложение».
  • Опрос-стикер → ДМ конвертит сильнее чем link-стикер: 12–18% взаимодействий vs <5% тапов. Схема: «Хочешь попробовать? Да/Нет» → кто ответил «Да», получает ДМ с предложением.
  • Не размывать голубой на декор/звёзды отзыва.
  • Не ставить link-стикер как единственный CTA — сначала прогреть опросом.

Алгоритм и охват

  • Instagram ранжирует приватные действия: сохранить, переслать, ответить — важнее публичных лайков.
  • Каждый макет должен провоцировать одно из трёх:
    → Сохранить — полезный контент (чеклист, лайфхак, «5 признаков»).
    → Переслать — эмоция + узнавание («это же про меня»).
    → Ответить в ДМ — открытый вопрос или опрос-стикер.
  • Репост сторис подписчиком — лучший сигнал алгоритму. Можно просить об этом прямо в тексте: «Поделись с подругой, которой это нужно».
  • Не публиковать контент без явного триггера для одного из этих действий — алгоритм не подхватит.

Чеклист перед отдачей

  • Каждый текст читается на своём фоне.
  • Цена/CTA — Jost по скриму, не сериф.
  • Один CTA, голубой только на пробном.
  • Мелкое ≥34px, критичное вне safe-zone, ≤15 слов на слайде.
  • Лого/айдентика на месте.
  • Один приватный триггер: сохранить / переслать / ответить.
  • Если цель — запись: опрос → ДМ, не сразу ссылка.
Не видно оффер за секунду — сила предложения не важна. Нет приватного триггера — алгоритм не разгонит. Читаемость → конверсия → охват.
