Типографика

Как набирать сторис, рилзы, посты и профиль Flow, чтобы читалось за секунду и работало на запись. Два шрифта, две роли — и ничего лишнего.

Рилзы (1080×1920)

  • Текст — в safe-zone: отступ сверху ≥250px (аватар/крестик), снизу ≥420px (подпись, кнопки, поле ответа).
  • Крупно и жирно: рилз смотрят в движении и со звуком — заголовок Playfair большой, ключевые слова Jost bold.
  • Вшитые субтитры — Jost Medium 44–52, белые по полупрозрачной плашке/скриму, не тоньше Medium.
  • Одна мысль на экран. Смена кадра = смена одной строки, а не абзаца.
  • Тонкий Light и сериф на мелком тексте по «живому» фону — не читается на ходу.

Посты и карусели (1080×1080 / 1080×1350)

  • Обложка карусели: крючок Playfair + подсказка «листай →» в углу.
  • Внутренний слайд: крупный номер (Playfair, градиент) + заголовок Jost caps + тело Jost.
  • Один акцентный цвет на слайд, остальное — белый/серый + воздух.
  • Длинный текст — лево-выключка (читается быстрее центра).
  • Больше двух шрифтовых ролей на макете и более 1 градиентного заголовка на кадр.

Один доминантный CTA на макет. Цена — крупнейший Jost-элемент после hero, никогда серифом по фото. Действие — в нижней трети (зона большого пальца).

Читаемость: текст по фото и градиенты (выстрадано)

  • Над фото текст — всегда светлый (`--on-photo`), независимо от темы. Тёмный текст темы на фото тонет.
  • Текст по фото — на скриме + тень. Для градиентного слова обычная `text-shadow` НЕ работает (текст обрезан по маске) — нужен `filter: drop-shadow()`.
  • Градиентный текст не заканчивается тёмным на тёмном: для текста — `--grad-text` (яркий «синий» конец `#4F7BFF`), для фото — `--grad-on-photo` (насыщенный, не пастель). Истинный `#0014FF` — только в заливках/CTA (`--grad-hero`).
  • Лого-watermark над фото — белое (`--logo-on-photo`) + тень.
  • Пастельный градиент по фото — washed out. Только насыщенный.
  • Текст по самому светлому участку фото без скрима.

Чеклист перед отдачей

Читаемость

  • Каждый текст читается на своём фоне (особенно над фото и на светлых участках).
  • Цена / время / CTA — Jost, не сериф по фото.
  • Лого / watermark виден.
  • Градиентный текст не уходит в тёмное на тёмном.
  • Мелкое ≥34px (1080).

Смысл и бренд

  • Один доминантный CTA на макет.
  • Критичное вне safe-zone (верх 250 / низ 420).
  • Айдентика на месте: бренд-градиент / аура / дуотон / лого — даже в капучино.
  • Тренд-2026: жирно, контрастно, без бледного пастеля.
  • Макс. 2 шрифт-роли + лого.

Коротко: Do / Don't

Do

  • Сериф — только эмоция (заголовок/цитата).
  • Jost — вся важная инфа, по скриму.
  • Кегль мелкого ≥34px (1080).
  • Один акцент на кадр.
  • CAPS только на коротких лейблах/названиях.

Don't

  • Абзацы серифом.
  • Тонкий Light на мелком.
  • Текст по фото без скрима.
  • 3+ шрифтовые роли.
  • Важное в safe-zone Instagram.
